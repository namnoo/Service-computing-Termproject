getPharmacyInfo
Request
GET http://localhost:8080/api/pharmacy/state_name/city_name
Parameters
Parameters Value Description
state_name string The full state name of Korea.
ex) 서울특별시, 충청남도
city_name string The full city name of Korea. It needs only ~gu name, unless the city is included in the ~do.
ex) 강남구, 종로구, 천안시 동남구, 천안시 서북구,
